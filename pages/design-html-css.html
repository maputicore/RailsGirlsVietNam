Phiên bản Tiếng Việt Trang chủ Sự kiện Blog Nhà Tài trợ Giới thiệu

1.Design header

Đặt đoạn mã sau vào dưới cùng của app/assets/stylesheets/application.css:

.navbar {
  min-height: 38px;
  background-color: #f55e55;
}

Bây giờ làm mới trang và kiểm tra các thay đổi. Bạn có thể thử thay đổi màu sắc hoặc phông chữ của tiêu đề. Bạn có thể kiểm tra các tài liệu tham khảo màu sắc http://color.uisdc.com/.

Coach： Nói về property display, inline và block.

Sau đó, hãy đặt các dòng ở phía dưới:

.navbar a.brand { font-size: 18px; }
.navbar a.brand:hover {
  color: #fff;
  background-color: transparent;
  text-decoration: none;
}

Coach: Giải thích 4 trạng thái của một liên kết

2. Design table

Chúng tôi chỉ đơn giản là sử dụng twitter Bootstrap, để đánh bóng table.find của chúng tôi dòng này từ app/views/ideas/index.html.erb và thay thế：

<table>

↓

<table class="table">

Thay đổi kích thước của hình ảnh bằng cách sử dụng các dòng sau.

<%= image_tag(idea.picture_url, width: 600) if idea.picture.present? %>

Thay đổi width và xem những gì sẽ xảy ra.

Thêm những dòng sau vào cuối tập tin app/assets/stylesheets/ideas.css.scss：

.container a:hover {
  color: #f55e55;
  text-decoration: none;
  background-color: rgba(255, 255, 255, 0);
}

Thử thêm một số background style với background-image tham khảo một số mẫu tại http://subtlepatterns.com/.

3.Thêm style cho footer

Thêm vào dưới cùng của app/assets/stylesheets/application.css những dòng sau:

footer {
  background-color: #ebebeb;
  padding: 30px 0;
}

Cố gắng thêm vào footer nhiều thứ hơn.

4.Thêm style vào button

Mở http://localhost:3000/ideas/new và tìm nút Create Idea thêm những dòng này vào.

app/assets/stylesheets/ideas.css.scss

.container input[type="submit"] {
   height: 30px;
   font-size: 13px;
   background-color: #f55e55;
   border: none;
   color: #fff;
 }

Coach： Giải thích làm thế nào để sử dụng border trong CSS, hãy thử thay đổi style của button như vòng quanh góc, thêm vào bóng tối hoặc màu sắc vv.
